# POS Pak De Surya

Frontend foundation untuk aplikasi point of sale modern, dibangun dengan React 19, Vite, Tailwind CSS, dan React Router.

## Menjalankan aplikasi

```bash
npm install
npm run dev
```

Buka `http://localhost:5173`. Login demo menerima username dan password apa pun selama keduanya diisi.

## Script

```bash
npm run dev
npm run build
npm run lint
npm run preview
```

## Struktur utama

```text
src/
├── assets/
├── components/
├── constants/
├── hooks/
├── layouts/
├── pages/
│   ├── Dashboard/
│   └── Login/
├── router/
├── services/
└── utils/
```

Alias yang tersedia: `@components`, `@layouts`, `@pages`, `@services`, dan `@assets`.
